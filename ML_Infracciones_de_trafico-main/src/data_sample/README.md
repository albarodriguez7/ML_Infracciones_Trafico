# data_sample

Muestra del dataset utilizado en el proyecto (máx. 5MB recomendado para el repositorio).

- En esta carpeta se incluye una **muestra** del dataset (`dataset_definitivo.csv`, primeras ~5000 filas) para poder ejecutar el código (máx. 5MB).
- Para el dataset completo, usar el repositorio original ML_Infracciones_de_trafico o colocar aquí el archivo completo si es público.
- El notebook principal `main.ipynb` cargará los datos desde `src/data_sample/dataset_definitivo.csv`.
- Variables clave: CUANTIA, PUNTOS, NUM_INFRACCIONES (para definir el target "alto riesgo"), más el resto de features para el modelo.
